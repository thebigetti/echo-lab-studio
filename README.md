Форма должна отправлять POST-запрос на /api/request.

Backend:
создай Express-сервер.
POST /api/request принимает name, contact, message.
Проверяет, что поля не пустые.
Отправляет сообщение в Telegram через Bot API:
https://api.telegram.org/bot<TOKEN>/sendMessage

TOKEN брать из TELEGRAM_BOT_TOKEN.
CHAT_ID брать из TELEGRAM_CHAT_ID.
Порт брать из PORT или 3000.

Добавить:
cors
express.json
express.urlencoded

Добавить GET /api/health, который возвращает:
{ "ok": true }

.env.example:
TELEGRAM_BOT_TOKEN=put_your_token_here
TELEGRAM_CHAT_ID=put_your_chat_id_here
PORT=3000
